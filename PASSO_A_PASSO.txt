SISTEMA DE PONTUAÇÃO FLL 2026 - FIREBASE

1) Criar projeto no Firebase
- Acesse https://console.firebase.google.com
- Criar projeto
- Adicionar app Web
- Copiar as configurações do Firebase

2) Ativar Firestore
- Firebase Console > Firestore Database
- Criar banco de dados
- Iniciar em modo de produção ou teste
- Região: escolha a mais próxima

3) Editar public/firebase-config.js
Substitua os campos COLE_AQUI pelos dados do seu projeto Firebase.

4) Regras do Firestore para teste inicial
Firebase Console > Firestore Database > Regras:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

ATENÇÃO: essa regra é aberta. Use apenas para teste/evento interno.
Depois podemos colocar login/senha para jurados.

5) Publicar no Firebase Hosting
Instale o Firebase CLI:
npm install -g firebase-tools

Faça login:
firebase login

Dentro da pasta robotica-score, rode:
firebase use --add
firebase deploy

6) Links do sistema
- Tela dos jurados: /index.html
- Dashboard ao vivo: /dashboard.html
- Administração: /admin.html

7) Regra do ranking
- Round TESTE aparece no dashboard, mas não vale ranking final.
- Ranking final usa a maior nota entre Round 1 e Round 2.
- Dashboard mostra Teste, Round 1, Round 2 e Melhor oficial.

8) Observação sobre discos de precisão
O sistema permite selecionar apenas uma opção de disco de precisão.

9) Observação sobre Missão 3
Parcial e total são opções exclusivas. Marcar uma desmarca a outra.

10) Atualização visual BIOGLOW
- O CSS foi atualizado usando como referência visual o livro de regras enviado.
- Foram adicionadas imagens em public/assets para cabeçalho, logo BIOGLOW e textura de fundo.
- A identidade usa verde principal, amarelo de destaque, pontos em vermelho e textura inspirada nas páginas de missões.

PUBLICAÇÃO IGUAL AOS OUTROS PROJETOS COM GITHUB + FIREBASE
1. Suba a pasta robotica-score para o GitHub.
2. No Firebase Console, acesse Hosting.
3. Clique em Get started / Começar.
4. Escolha a opção de integração com GitHub.
5. Selecione o repositório do projeto.
6. Em public directory, use: public
7. Deploy automático: mantenha ativado.
8. Depois disso, sempre que você enviar alteração para o GitHub, o Firebase atualiza o site sozinho.
